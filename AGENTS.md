# Project Rules

This project is a migration from Streamlit to FastAPI + React.

See [README.md](./README.md) for how to run the stack and where business logic lives.

Goals

- readable code
- maintainable
- dockerized
- production ready

Rules

- Keep code simple.
- Never overengineer.
- Never write business logic inside React.
- Never duplicate code.
- One responsibility per function.
- Use descriptive names.
- Use Docker.
- Use environment variables.
- Keep files small.
- Explain every major change.
- Preserve original behavior.
